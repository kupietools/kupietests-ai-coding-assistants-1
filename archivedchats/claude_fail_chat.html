    The error occurs because PHP is trying to interpret the JavaScript template literal syntax as PHP code. When using PHP heredoc syntax (<<<JS), PHP tries to parse the content and interpolate variables.

    To fix this, you need to escape the JavaScript template literals. Here's how to modify the relevant part of the code:
